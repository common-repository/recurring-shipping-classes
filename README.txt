== Description ==

This plugin works with WooCommerce. It allows you to change shipping classes dynamically according to dates.

You can choose the recurring month & day, and it will be defined for every year.

Once the dates of your shipping classes are defined, you can for example, assign different shipping costs to them.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Make sure that WooCommerce is enabled and activate the plugin through the 'Plugins' menu in WordPress
3. Edit a product, then choose "Recurring Shipping Class" tab

== Frequently Asked Questions ==

= When my product will be updated with the shipping class ? =

When you save your product, the shipping class will be updated. After that, every hour, the shipping class will be checked, and updated if needed.
